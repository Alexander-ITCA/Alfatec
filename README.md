## Mantenimiento

- Para cambiar imágenes de galería, reemplazar archivos dentro de `img/galeria/` conservando nombres o actualizando rutas en `galeria.html`.
- Para cambiar el spot, reemplazar `video/spot.mp4`.
- Para editar colores, modificar variables CSS en `css/styles.css`.
- Para ajustar navegación, formularios o video, editar `js/main.js`.
